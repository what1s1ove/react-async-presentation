React Hooks

Hooks are a new addition in React 16.8. They let you use state and other React features without writing a class.

        const Counter = () => {
          const [count, setCount] = React.useState(0);
          return ( 
            <div> 
              <p>You clicked {count} times</p> 
              <button onClick={() => setCount(count + 1)}> 
                Click me 
              </button> 
            </div> 
          ); 
        }
reactjs.org/hooks-intro

Why Hooks

  • Simplify entry threshold (OOP, this)
  • Minimize app code (visually + bundle size)
  • Reuse code and void HOC Hell
  • Add missing features to functional components:
    • State
    • Side effects
    • Lifecycle methods

How Hooks Work

  • Hook is a function. The function is called every time the component is rendered
  • React tracks calls of these functions
  • Hook is a common function but with a bit of magic

Hooks Limitations

  • Cannot be used in class components
  • The sequence of Hook is important. Cannot be used in conditions, loops
  • Hooks can only be used in components or custom hooks
  • No support for all lifecycle methods:
    • getSnapshotBeforeUpdate
    • getDerivedStateFromError
    • componentDidCatch

Hooks API

  • useState
  • useEffect
  • useCallback
  • useMemo and etc.
reactjs.org/hooks-api

Custom Hooks

        const useModalState = (initialOpen = false) => {
          const [isOpen, setIsOpen] = useState(initialOpen); 
         
          const handleOpen = () => setIsOpen(true); 
          const handleClose = () => setIsOpen(false); 
         
          return { 
            isOpen, 
            handleOpen, 
            handleClose, 
          }; 
        };

Redux Middleware

        // src/store/middlewares/logger/logger.middleware.js
        const logger = (store) => (next) => (action) => {
          console.log('state before', store.getState());
          console.log('dispatching', action);
          next(action);
          console.log('state after', store.getState());
        };
      
        // src/store/store.js
        import { createStore, applyMiddleware } from 'redux';
        import { logger as loggerMiddleware } from '../middlewares/middlewares.js';
         
        const store = createStore(reducer, applyMiddleware(loggerMiddleware));
